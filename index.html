기대효과

✅ 실제 가능한 것들

1. 프로젝트 전체 흐름 이해

  • 기획부터 배포까지 전 과정을 경험하며 큰 그림 파악
  • 각 기술이 어떻게 연결되는지 구조적 이해

2. AI 도구 활용 능력

  • Claude Code와 효과적으로 협업하는 프롬프팅 능력
  • 에러 해결 과정에서 AI를 활용하는 실무 역량
  • 빠른 프로토타이핑 및 아이디어 구현 능력

3. 시간 대비 높은 완성도

  • 전통적 학습 대비 1/5 시간으로 실무급 프로젝트 완성
  • 문법 암기 대신 개념 이해와 구현에 집중
  • 즉각적인 피드백으로 빠른 학습 사이클

4. 실전 포트폴리오 확보

  • 취업/이직에 사용 가능한 실제 배포된 웹사이트
  • GitHub 저장소 관리 경험

⚠️ 한계점

1. 거대 프로젝트는 아직 어려움

복잡한 엔터프라이즈급 시스템 구축에는 한계가 있으며, 다수의 마이크로서비스 연동 같은 대규모 아키텍처는 부적합합니다.

2. 세밀한 커스터마이징 제약

매우 특수한 알고리즘이나 최적화가 필요한 경우 직접 코딩이 필요하며, 픽셀 단위 디자인 조정 등 세부 작업은 여전히 시간이 소요됩니다.

3. 깊은 기초 지식 부족 가능성

자료구조, 알고리즘 같은 CS 기초는 별도 학습이 필요하며, 코드 한 줄 한 줄의 의미를 완벽히 이해하기 어려울 수 있습니다.

4. AI 의존도

AI 없이는 같은 수준의 개발 속도 유지가 어려우며, 네트워크 오류 등 AI 사용 불가 시 생산성이 급감할 수 있습니다.
